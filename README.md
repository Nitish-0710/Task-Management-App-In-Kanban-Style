## Installation

1. Clone the repository:
   ```bash
    git clone https://github.com/yourusername/kanban-app.git
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Run development server:
    ```bash
    npm start
    ```

4. Build for production:
    ```bash
    npm run build
    ```
